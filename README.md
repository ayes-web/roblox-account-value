<h1 align="center">Roblox account value</h1>

<p align="center">WebApp that displays account value in robux and euro</p>

### Build
```
nix build
```

### Shell
```
nix develop
```

## Makefile usage

### Build

```
make
```

### Start dev server

```
make dev
```

### Preview production build

```
make start
```
